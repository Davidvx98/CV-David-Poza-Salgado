# Sistema de Internacionalización (i18n)

Este proyecto ahora soporta dos idiomas: **Español (ES)** e **Inglés (EN)**.

## 🌍 Idiomas disponibles

- **Español (ES)** - Idioma predeterminado
- **English (EN)**

## 📁 Estructura

```
src/
├── i18n/
│   ├── es.json          # Traducciones en español
│   ├── en.json          # Traducciones en inglés
│   └── utils.ts         # Utilidades i18n
├── pages/
│   ├── index.astro      # Página principal (ES)
│   └── en/
│       └── index.astro  # Página en inglés
└── components/
    └── LanguageSelector.astro  # Selector de idioma
```

## 🔗 URLs

- **Español (predeterminado):** `/` o `/index.html`
- **Inglés:** `/en` o `/en/index.html`

## 🎨 Selector de idioma

El selector de idioma está ubicado en el header y permite cambiar entre español e inglés en cualquier momento. Al cambiar de idioma, se mantiene la misma sección de la página.

## ✍️ Cómo añadir nuevas traducciones

1. Abre `src/i18n/es.json` y añade tu nueva clave de traducción:
```json
{
  "miSeccion": {
    "titulo": "Mi título en español"
  }
}
```

2. Añade la misma clave en `src/i18n/en.json`:
```json
{
  "miSeccion": {
    "titulo": "My title in English"
  }
}
```

3. Usa la traducción en tu componente:
```astro
---
import { getLangFromUrl, useTranslations } from '../i18n/utils';

const lang = getLangFromUrl(Astro.url);
const t = useTranslations(lang);
---

<h1>{t('miSeccion.titulo')}</h1>
```

## 🛠️ Funciones disponibles

### `getLangFromUrl(url: URL): Lang`
Obtiene el idioma actual desde la URL.

### `useTranslations(lang: Lang)`
Retorna una función `t()` para obtener traducciones.

### `translatePath(path: string, lang: Lang): string`
Convierte una ruta a su versión traducida.

## 📝 Notas

- El español es el idioma predeterminado y no tiene prefijo en la URL
- El inglés usa el prefijo `/en`
- Todas las traducciones están centralizadas en los archivos JSON
- El sistema está configurado en `astro.config.mjs`
